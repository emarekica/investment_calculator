# Investment Calculator

A lightweight React application built with Vite.
Focused on demonstrating core investment calculation UIs and testing fundamentals (component and unit tests).
Primarily intended as a developer reference for modern front-end development and test coverage best practices.

## Project Overview

This repository provides:

- A React app scaffolded with Vite
- Core investment calculator UI logic
- Demonstrations of component and integration tests

The goal is to provide developers with an example of how to structure a front-end project with a solid testing strategy.

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| UI | React |
| Build | Vite |
| Testing | Jest, React Testing Library |
| Package Manager | npm / yarn |

Actual dependencies and scripts are defined in `package.json`.
